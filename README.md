# flutter_lead_management_app
This Flutter app is a Mini Lead Manager. Users must perform CRUD operations: Add, View, Edit, Delete leads. Each lead requires Name and Contact details. Status (New, Contacted, Converted, Lost) must be trackable and filterable . Persistent storage (SQLite preferred) and robust architecture (Provider used) are mandatory.
